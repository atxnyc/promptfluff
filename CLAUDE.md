# promptfluff

A small, fun joke project: a Claude Code encouragement hook (`src/`, mirrored in `plugin/`) plus its marketing site (`site/`). Keep the effort proportional to the joke.

## Working style

- **Do not use git worktrees in this repo.** Edit in place on `main` (or a simple branch), commit, done. Worktrees are overkill for what is almost always a one- or two-file tweak.
- Don't over-engineer. No elaborate isolation, multi-agent workflows, or heavy process for small content/copy changes. Match the ceremony to the task — which is usually "none."
